                                      Day 03 – Social Media User Analysis Dataset

. Data Source : https://www.kaggle.com/datasets/rockyt07/social-media-user-analysis

. Dataset Description:
  This dataset contains information about social media users, including demographics, lifestyle, and usage patterns.

. Type of Data:
  Structured data (tabular data)

. ML Concept:
  Machine Learning – analyzing and predicting user behavior.

. Problem Statement:
  Can we analyze and predict user behavior based on social media usage and demographic features?

. Input:
  Age, gender, country, screen time, followers, likes, income level, education, etc.

. Output:
  User engagement level / behavior prediction.

. What we learned:
  - Younger users spend more time on social media.
  - Higher screen time may indicate addiction.
  - Lifestyle and demographics affect social media usage.

. Real-life use:
  Used by social media companies and marketers to understand user behavior and improve recommendations.
