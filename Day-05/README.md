                                                Day 05 – House Price Prediction Dataset
                                                 
. Dataset Source: https://www.kaggle.com/datasets/shree1992/housedata

. Dataset Description:
  This dataset contains real estate information such as house price, number of bedrooms, bathrooms, living area size, land area, floors, and house condition.
  
. Type of Data:
  Structured data (Tabular data)
  
. ML Concept:
  Machine Learning – Regression (predicting continuous values).
  
. Problem Statement:
  Can we predict the price of a house based on its features like size, location, and number of rooms?
  
. Input (Features):

   - Number of bedrooms
   - Number of bathrooms
   - Living area size (sqft)
   - Lot size
   - Number of floors
   - Waterfront view
   - House condition

. Output (Prediction):
  House price
  
. What We Learned:

   - Bigger houses usually have higher prices.
   - Location and waterfront view increase house value.
   - Number of bedrooms and bathrooms affect pricing.
   - House condition plays an important role in valuation.

. Real-Life Use:

  - Used by real estate companies
  - Property websites
  - Banks
  - Builders to estimate house prices
  - Approve home loans
  - Analyze market trends.
