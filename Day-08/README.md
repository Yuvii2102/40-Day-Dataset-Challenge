                                   Day 08 - Customer Behaviour Shopping Analysis
               
* Dataset Source : https://www.kaggle.com/datasets/anishdevedward/loan-approval-dataset

* Dataset Description
  -     This dataset contains customer shopping information including demographics, product categories, purchase amount, location, and preferences. It helps analyze how customers shop and spend.
 
* Type of Data
 
  Structured data (tabular retail/customer dataset)

* ML Concept
  
  Exploring Data Analysis + Customer behavior prediction using Machine Learning.

* Problem Statement
  
  Can we analyze customer shopping patterns and spending behavior to understand purchasing trends?

* Input
  
  - Age
 
  - Gender
 
  - Item purchased
 
  - Category
 
  - Purchase amount
 
  - Location

  - Size
 
  - Color
 
  - Season
 
* Output
  
  - Spending pattern insights
  
  - Customer behavior trends
  
  - Purchase preferences prediction

* What We Learned
  
  - Different age groups show different shopping patterns
   
  - Certain product categories attract more spending
  
  - Seasonal trends affect purchases
  
  - Demographics influence buying behavior
 
* Correlation Insight
  
  - Age vs purchase amount shows spending trends

  - Category vs purchase frequency reveals popular items

  - Season vs category highlights seasonal demand
 
* Used by:
  
  - Retail companies to improve sales strategy
   
  - Marketing teams for targeted promotions
   
  - Inventory planning based on demand
