                              😴 Day 28 – Student Sleep Patterns Analysis & Sleep Quality Prediction

✨Dataset Source

https://www.kaggle.com/datasets/arsalanjamal002/student-sleep-patterns

---

## 📌 Dataset Description

This dataset contains information about university students and their sleep habits.

Total Records:
500 students

Each row represents one student and includes:

- Sleep duration
- Study hours
- Screen time
- Caffeine intake
- Physical activity
- Sleep quality rating (1–10)

⚠ Note: This is synthetic data (generated artificially).

---

## 📊 Type of Data

Structured dataset  
Lifestyle + Sleep related data  

---

## 🤖 ML Concept

Machine Learning – Regression  

We predict:

Sleep Quality (1 to 10)

Where:
- 1 → Very poor sleep
- 10 → Excellent sleep

---

## 🎯 Target Column

Sleep_Quality

---

## 📥 Important Input Features

- Sleep_Duration → Hours of sleep per night  
- Study_Hours → Hours spent studying  
- Screen_Time → Hours spent on phone/laptop  
- Caffeine_Intake → Number of caffeinated drinks  
- Physical_Activity → Minutes of exercise  
- Weekday_Sleep_Start  
- Weekend_Sleep_Start  

---

## 📤 Output

Predicted Sleep Quality Score (1–10)

---

## 🧠 What We Learned

- More sleep duration → Better sleep quality  
- High screen time → Lower sleep quality  
- More caffeine → Poor sleep  
- Physical activity → Improves sleep  
- Late sleep time → Reduces sleep quality
