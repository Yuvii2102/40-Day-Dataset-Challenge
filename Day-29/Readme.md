                                                  🎓 Day 29 – Online Learning Behavior Dataset

✨ Dataset Source 

https://www.kaggle.com/datasets/maazshaikh05/online-learning-behavior-dataset

---

📌 Dataset Description

This dataset contains information about students who use online learning platforms.  
It includes details like age, country, education level, platform used, device used, and learning activity.

The dataset helps us understand how students learn online and how engaged they are in courses.

---

📊 Type of Data

Structured data (online learning activity dataset)

---

🤖 ML Concept

Machine Learning – Classification / Prediction

Used to predict student satisfaction or course completion based on learning behavior.

---

❓ Problem Statement

Can we analyze student learning behavior and predict their satisfaction or course completion using their activity data?

---

📥 Input

- Age  
- Gender  
- Country  
- Education Level  
- Field of Study  
- Platform Used (Coursera, Udemy, edX, etc.)  
- Device Used (Mobile, Laptop, Tablet, Desktop)  
- Learning Mode (Self-Paced, Instructor-Led, Hybrid)  
- Daily Learning Hours  
- Quizzes Attempted  
- Assignments Submitted  

---

📤 Output

- Course completion rate  
- Student satisfaction score

---

🧠 What We Learned

- Students using laptops learn more hours  
- Self-paced learning is very common  
- Engagement increases when students complete quizzes and assignments  
- Platform choice affects learning patterns

---

🔗 Correlation Insight

- Learning hours ↔ course completion  
- Assignments submitted ↔ satisfaction score  
- Platform used ↔ engagement level

---

📈 Visualization Ideas

- Device usage graph

- Completion rate comparison
